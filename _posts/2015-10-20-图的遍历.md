###DFS（Depth First Search）

```
void DFS( Vertex V)
{
	visited[V] = true;
	for ( V的每个邻接点W )
		if ( !visited[W])
			DFS(W);
}
```

邻接矩阵 找邻接点 每行扫一次，一共N行 O（N^2)


###BFS (Breadth First Search)


```
void BFS(Vertex V)
{
	visited[V] = true;
	Enqueue(V,Q);
	while(!IsEmpty(Q)){
		V = Dequeue(Q);
		for (V的每个邻接点W)
			if (!visited[W]){
				visited[W] = true;
				Enqueue(W,Q);
			}
	}
}
```

时间复杂度和DFS一样
