###再种一棵树

###表示

来看一颗常见的树


```
		  56
		 /\ \
		/  \ \ 
	   /    \ \ 
	  93   68  75
	       |
	       76  
```

为了节约空间，这种树我们一般的表示会是



```
struct node{
  int data;
  node* child;
  node* sibling;
}

```

这样的一棵树，会是这样：


```
56 
|
⌄
93 -> 68 -> 75
      |
      ⌄
      76
       
```



其它部分都是指零的
